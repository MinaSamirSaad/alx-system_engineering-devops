# Shell Basics

## About this Directoy

This directory contains programs that demonstrate make use of the following linux commands.

1. cd
2. ls
3. pwd
4. less
5. file
6. ln
7. cp
8. mv
9. rm
10. mkdir
11. type
12. which
13. help
14. man
